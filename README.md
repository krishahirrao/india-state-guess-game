# India States & Union Territories Guessing Game

An interactive Python game that challenges players to guess all **28 States** and **8 Union Territories** of India.

As you correctly guess a State or Union Territory, its name is displayed on the map in real time. If you get stuck, simply type **Exit** and the game automatically generates a **Remaining_States.csv** file containing all the States & UTs you couldn't guess, making it easy to review and learn later.

---

## 🎯 Features

- 🗺️ Interactive map of India using Turtle Graphics
- 🇮🇳 Guess all **36 States & Union Territories**
- ✅ Correct guesses appear instantly on the map
- 📊 Live score tracker
- ⌨️ Simple and interactive text input
- 📄 Type **Exit** anytime to generate a **Remaining_States.csv** file
- 📚 Review the generated CSV to learn the States & UTs you missed
- 🎮 Beginner-friendly project for learning Python and Indian geography

## 🎮 How to Play

1. Launch the game.
2. An outline map of India will appear.
3. Enter the name of any Indian State or Union Territory.
4. Every correct guess will appear at its corresponding location on the map.
5. Continue guessing until you've named all **36 States & Union Territories**.
6. If you don't know the remaining answers, simply type:

```
Exit
```

7. The game will automatically create a file named:

```
Remaining_States.csv
```

The generated CSV contains every State & Union Territory you didn't guess, making it perfect for revising before your next attempt.
